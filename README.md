# Interio Backend

This is the backend API for the Interio Decoration E-Commerce Website, built with:

- Express.js
- MongoDB & Mongoose
- JSON Web Tokens (JWT)
- CommonJS Modules

## Features

- User Authentication (JWT)
- Product Management
- Category Management
- Shopping Cart
- Orders & Checkout
- Reviews System

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run in development

```bash
npm run dev
```

3. Create a `.env` file

```
PORT=5000
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret_key
NODE_ENV=development
```
